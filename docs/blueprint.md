# **App Name**: BasaTranslator

## Core Features:

- Input Text Area: User input field for text in Indonesian or Sundanese.
- Language Selection: A dropdown to select the input language (Indonesian or Sundanese).
- Translation Engine: Translation engine powered by a trained model with 1000 corpus, handling translation between Indonesian and Sundanese.
- Output Text Area: Displays the translated text in the target language.
- Translate Button: Button to initiate the translation process.

## Style Guidelines:

- Primary color: Soft green (#A8DF8E) to represent growth and language.
- Secondary color: Light beige (#F0EAD6) for a clean and neutral background.
- Accent: Terracotta (#E07A5F) for buttons and interactive elements to reflect Indonesian/Sundanese culture.
- Clear and readable font for input and output text.
- Simple icons for language selection and other functions.
- Clean and intuitive layout for easy use.
